## 🌐 What is NJ SecForge?

**NJ SecForge** is an interactive cybersecurity learning platform designed for developers to understand web vulnerabilities through **hands-on practice** instead of theory alone.

Unlike traditional tutorials, NJ SecForge lets you:

- 🔍 Explore real vulnerabilities
- 💥 Perform safe attacks
- 🛡 Compare vulnerable vs secure code
- 🧠 Understand *why* exploits work

> Think like an attacker. Build like a defender.

✅ OWASP Top 10 (2025) Interactive Labs

## 🧠 Learning Philosophy

Each lab teaches security using **three perspectives**:

> Vulnerable Code → Attack Simulation → Secure Fix

You don't just learn *what* happened — you learn:

- Why it happened
- How attackers think
- How developers prevent it

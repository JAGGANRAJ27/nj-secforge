# 🔥 NJ SecForge

Interactive OWASP Top 10 learning platform for developers.

Learn security by:
- Understanding vulnerabilities
- Practicing real attacks
- Applying secure coding fixes

---

## 🚀 Features

✅ OWASP Top 10 Labs  
✅ Vulnerable vs Secure comparison  
✅ Developer-focused explanations  
✅ Open-source learning platform

---

## 🧪 Available Labs

| Lab | Status |
|-----|--------|
| SQL Injection | ✅ |
| XSS | 🚧 Coming Soon |
| Broken Auth | 🚧 Coming Soon |

---

## ⚙️ Run Locally

```bash
git clone https://github.com/JAGGANRAJ27/nj-secforge
cd nj-secforge
npm install
node server.js

Open http://localhost:3000 in your browser
```
